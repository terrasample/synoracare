# SynoraCare AI Mobile App

Expo React Native app for SynoraCare AI.

## Run locally

```bash
cd mobile-app
npm install
npm run ios
# or
npm run android
# or
npm run web
```

## What it does

- Shows SynoraCare AI branded launch screen
- Checks backend health (`/health`)
- Opens live production console inside an in-app WebView

## Production endpoints in app

- Web: `https://www.synoracare.com`
- API health: `https://synoracare-backend.onrender.com/health`
